#  Strong Hold - Landing Page
Clique aqui para ver o site https://francisco-caio.github.io/strong-hold/

Landing page responsiva para academia de musculação, crossfit e artes marciais.

##  Tecnologias
- HTML5 semântico
- Tailwind CSS (via CDN)
- Flowbite Components
- Font Awesome Icons
- JavaScript vanilla (dark mode, menu mobile)

##  Desenvolvimento Mobile
Projeto codificado 100% em ambiente mobile:
- Tablet Android + teclado e mouse
- Editor: Spck Editor
- Navegador para preview e pesquisa

## Funcionalidades
- Design responsivo (mobile-first)
- Dark/Light mode toggle com persistência
- Menu mobile hambúrguer
- Navegação suave entre seções
- Cards de planos interativos

##  Créditos de Imagens
- Avatares da equipe: Flowbite (placeholders)
- logo: canva(sandbox)
- Fotos de treino: [Unsplash.com - sandbox ]
- Ícones: Font Awesome
